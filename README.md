# mi-primer-pr

Este repositorio es un espacio de práctica para aprender a contribuir en GitHub mediante pull requests.

## ¿Qué es un pull request?

Un pull request (PR) es una propuesta de cambio en un repositorio. Permite revisar el código antes de integrarlo en la rama principal.

## Cómo contribuir

1. Haz un fork del repositorio o crea una rama nueva.
2. Realiza tus cambios.
3. Abre un pull request describiendo qué cambiaste y por qué.
